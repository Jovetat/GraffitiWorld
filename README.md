# GraffitiWorld

## 简介

个人项目：旨在通过手势识别完成自由涂鸦、并配套分享社区

前端：Vue 3、handtrack.js、Ant Design Vue

后端：Node.js

### 预计完成功能

>   #### 手势绘画
>
>   >   point:画笔
>   >
>   >   open:橡皮
>   >
>   >   closed:撤销
>   >
>   >   pinch:菜单
>
>   #### 菜单
>
>   >   第一行：画笔样式切换
>   >
>   >   第二行：画笔颜色切换
>   >
>   >   第三行：保存、分享、我的画作、退出
>   >
>   >   os：悬浮在指定样式或颜色后完成切换
>   >
>   >   切换手势时退出悬浮二级菜单
>
